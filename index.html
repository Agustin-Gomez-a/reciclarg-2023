A continuación, Veremos contenedores de reciclaje según su color:


Contenedor Amarillo

Contenedor Amarillo

El contenedor amarillo es el adecuado para reciclar plásticos, latas y envases. En este tipo de contenedores se debe almacenar todo material que esté hecho a base de plástico. Como botellas de plástico, envases de alimentos, bolsas de plástico, briks de leche, etc. También las latas de conserva y de refrescos deben depositarse en el contenedor amarillo




Contenedor Verde

Contenedor verde

El contenedor verde, también conocido como iglú verde, es para depositar vidrio. En él deben reciclarse botellas de vidrio, tarros, espejos y cristales rotos, pero no cerámica, porcelana, metales ni plásticos. Antes de tirar una botella o tarro de vidrio, hay que quitarle el tapón y reciclarlo en el contenedor correspondiente para metal, plástico o corcho.





Contenedor azul

Contenedor azul

El contenedor azul es para depositar papel y cartón, incluyendo cajas, envases de cartón, periódicos, revistas, documentos, entre otros. Se debe plegar el cartón para optimizar el espacio del contenedor.
